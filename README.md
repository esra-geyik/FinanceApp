# 💰 FinanceApp

FinanceApp, kullanıcıların gelir ve giderlerini yönetebilmesini sağlayan Java tabanlı bir finans takip uygulamasıdır. Uygulama, nesne yönelimli programlama (OOP) prensipleri ve katmanlı yapı yaklaşımı kullanılarak geliştirilmiştir.

---

## 🎯 Projenin Amacı

Bu proje, temel finansal işlemlerin dijital ortamda yönetilmesini sağlamak ve Java programlama dili ile OOP mimarisinin pratik bir uygulamasını geliştirmek amacıyla hazırlanmıştır.

Ayrıca proje, yazılım geliştirme süreçlerinde:
- Modüler yapı oluşturma
- Kod okunabilirliği
- Sorumlulukların ayrılması (Separation of Concerns)

gibi temel yazılım mühendisliği prensiplerini uygulamayı hedefler.

---

## ⚙️ Uygulama Özellikleri

- Kullanıcı giriş sistemi (Authentication)
- Gelir ve gider kayıt işlemleri
- Kategori bazlı işlem yönetimi
- Kullanıcıya özel işlem geçmişi
- Bakiye hesaplama
- Tarih bazlı kayıt takibi (LocalDate kullanımı)

---

## 🧱 Kullanılan Teknolojiler

- Java
- Object-Oriented Programming (OOP)
- ArrayList veri yapısı
- Java Time API (LocalDate)
